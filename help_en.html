Multisensory Road Condition Analysis

Version 2.01

This application lets you load ROS 2 bag files and inspect IMU signals together with synchronized GPS tracks, videos and point clouds. Use the menu bar to open a bag and select the IMU topics you want to analyse.

Map and Videos

The map tab displays the GPS track with optional heading arrows. Peaks detected in the acceleration signal are marked by black star icons. Clicking on a star opens the video/point cloud tab and shows a short clip around that time. You can also click star markers in the plots to jump directly to the corresponding video segment.

Settings Parameters

ParameterMeaning
Mounting OverridesManually specify sensor → vehicle rotation matrices.
Rotation ModeChoose between stored overrides or automatic alignment.
Peak DetectionConfigure ISO peak threshold, minimum distance and whether only the maximum peak is kept.
Weighting ModeToggle between comfort (default) and health weighting.

Exported CSV/JSON

The Export CSV command writes a CSV file with all processed signals and labels. A simplified example is shown below:

time,time_abs,accel_x,accel_y,accel_z,speed_mps,accel_corr_x,accel_corr_y,accel_corr_z,label_name
0.00,1700000000.00,0.04,0.02,9.81,0.00,0.04,0.02,-0.05,unknown

A JSON file with the same stem stores meta information such as sensor model and rotation matrix:

{
  "file_format": "imu_v1",
  "sensor_topic": "/imu0",
  "sensor_fs_accel_g": 16,
  "rotation_available": true,
  "vehicle_rot_mat": [[1,0,0],[0,1,0],[0,0,1]],
  "sampling_rate_hz": 50.0
}
