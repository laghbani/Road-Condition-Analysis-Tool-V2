Acceleration Signals

NameDescription
Raw AccelerationDirect measurements from the IMU.
g-corrected AccelerationAcceleration corrected for gravity.
Vehicle Frame AccelerationAcceleration rotated into the vehicle frame.
ISO WeightedAcceleration filtered using ISO 2631‑1 weighting.

Map and Videos

The map tab displays the GPS track with optional heading arrows. The video/point cloud tab visualizes recorded frames and point clouds synchronized with the acceleration data.

Settings Parameters

ParameterMeaning
Mounting OverridesManual rotation matrices for each sensor.

CSV File Structure

ColumnDescription
timeTime relative to the start of the bag.
time_absAbsolute ROS2 timestamp.
accel_x/y/zRaw acceleration components in sensor axes.
speed_mpsInterpolated vehicle speed from GPS.
gyro_x/y/z(Optional) Angular rate in rad/s.
accel_corr_x/y/zAcceleration after gravity compensation.
grav_x/y/zEstimated gravity vector.
accel_veh_x/y/z(Optional) Acceleration in the vehicle frame.
|a|_rawMagnitude of raw acceleration.
|ω|_rawMagnitude of angular rate.
clipped_flag1 if the sensor was saturated.
label_idNumeric identifier of the label.
label_nameTextual label of the segment.
